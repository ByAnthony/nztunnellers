## Contents

## Brief

The New Zealand Tunnellers Web App is dedicated to the men of the New Zealand Tunnelling Company who fought beneath the no man's land during the First World War.

## Dependencies

The project uses:

* Python 3;
* Pytest
* Flask;
* Flask_MySQLdb;
* MySQL.
